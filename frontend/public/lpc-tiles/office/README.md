## Usage Notes

- All tiles align to the standard 32x32 LPC grid
- Multi-tile objects (desk, TV, sink) span multiple 32px cells — place them accordingly
- The Copy Machine Copy Light is an 8-frame animation strip
- The TV has animation states: off, color bars, and 3 static/noise frames
- Laptop has open/closed states useful for showing active/inactive workstations
- Credits.txt contains per-asset attribution details
